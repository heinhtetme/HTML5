Day 01 - About Web Development

Lecture 01

Modern Web design has three pillars:
- HTML (Hypertext Markup Language) - Structuring web page (in 1989)
- CSS (Cascading Style Sheets) - Designing web page (in 1996)
- JS (JavaScript) - Interactive front end web development (in 1995)

What do you need to make Websites (or apps)?
- Code editor (eg. NotePad++, sublimeText, Visual Studio Code)
- Web Browser (eg. Firefox, Google Crome)

Two types of web developers:
- Web Designer - Graphics ($$)
- Web Developer - Coding ($$$$)
